# Hybr1d_backend

## Project Description
This project has build APIs for a e-commerce platform in  NodeJS.
The API support features like getting a user registered , user login , google oAuth , facebook auth , create order , create seller catalog , getting list of seller products. 	
## Technologies
Project is created with:
* Node.js
* Express.js
* MOngoDB as Database 
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../Hybr1d_backend
$ npm install
$ npm start
```
